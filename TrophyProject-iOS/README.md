# Trophy Project iOS

Native SwiftUI iPhone app for Trophy Project.

## Open In Xcode

1. Open `TrophyProject.xcodeproj`.
2. Select the `TrophyProject` scheme.
3. Choose an iPhone simulator or device.
4. Build and run.

The app uses the live Trophy Project endpoints at `https://trophyproject.pro`:

- `/api/games.php`
- `/api/game-detail.php`
- `/api/planner-search.php`
- `/api/planner-title.php`
- `/api/psn-user-search.php`
- `/api/psn-user-titles.php`
- `/api/psn-user-title.php`

Planner data is stored locally on-device with `UserDefaults`.
